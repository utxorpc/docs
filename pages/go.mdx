# Go SDK

The UtxoRpc Golang SDK is a comprehensive toolkit for building clients and servers to interact with UTxO-based blockchains using the UtxoRpc protocol. This SDK simplifies the process of developing Golang applications that communicate with blockchain gateways, providing an easy-to-use interface for querying and managing transactions.

## Prerequisites

- Go 1.16 or higher
- `go`, the Go command line tool

## Installation

To add the UtxoRpc Golang SDK to your project, run the following command:

```bash
go get -u github.com/utxorpc/go-sdk
```

## Usage

```go
package main

import (
	"github.com/utxorpc/go-sdk"
)

func main() {
	// Code to interact with the UtxoRpc API
}
```
